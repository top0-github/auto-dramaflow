# 分镜提示词风格 · 赛博朋克黑色电影 · 技法参考

## prompt 结构

```
[风格标签] + [时长] + [时间编码镜头序列] + [技术参数]
```

### 风格标签模板
```
Style: Cyberpunk Noir, Blade Runner 2049 cinematography, 35mm anamorphic.
```

### 镜头描述模板
```
[时间] Shot [N]: [景别/运镜]. [场景]. [角色动作]. [光影细节]. [大气介质].
```

## 景别策略

| 景别 | 占比 | 功能 |
|------|------|------|
| 大远景 | 15% | 建立城市压迫感——渺小人物 vs 巨型建筑 |
| 中景 | 35% | 角色在环境中的互动——走过夜市/穿过蒸汽 |
| 近景 | 30% | 面部被霓虹光斑照亮——情绪锚点 |
| 大特写 | 20% | 眼球反射全息屏/义体机械细节/雨滴滑落 |

## 运镜策略

- 缓慢推轨（Slow Dolly）：建立空间，逐步揭示巨型结构
- 手持微晃：角色探索/紧张段落
- 无人机航拍：城市全貌定场
- 慢动作：关键时刻——雨滴/弹壳落地/霓虹闪烁

## 光影关键词库

`volumetric lighting`, `god rays through fog`, `neon reflections on wet asphalt`,
`cyan ambient + magenta accent`, `low-key lighting`, `practical light sources`,
`atmospheric haze`, `anamorphic lens flare`, `film grain 10%`,
`shallow depth of field`, `bokeh neon background`

## 禁止

- 平光/均匀照明/无阴影的"干净"画面
- 无大气介质的"真空"画面（必须有雾/烟/蒸汽/雨）
- 自然光为主光源的镜头
