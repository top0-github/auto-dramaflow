# 赛博朋克场景生成 prompt 规范

## 场景类型

### 废弃海上城市（megastructure）

```
一座广袤的废弃海上城市在黄昏中，巨型未知用途的建筑从海洋中升起，
形成错落的剪影。骨白色单体塔楼与藤壶覆盖的钢铁融合，
独眼巨人般的环形结构以断裂的角度倾斜，
锈蚀的骨架式龙门架挂着死去的线缆。
黑色暗涌在桥墩之间翻滚，沉船半埋在塔脚。
浓厚的海雾紧贴底部，而上层结构刺入青紫色伤痕天空。
远处塔楼高处零星闪烁的微弱灯光像遥远的眼睛。
情绪化低主光照明——阴天天空的冷青色环境光，
远处建筑漏出的暖琥珀钠灯光勾勒剪影，
低角度透过前景锈蚀望向建筑群，
35mm变形宽银幕镜头，中等广角，浅景深，
前景锈蚀虚焦。细密的大气雾霾将远处建筑压成层次分明的剪影。
海洋调色板：青蓝/骨白/铁锈，点缀远处的暖色光点。
胶片颗粒，高对比度，编辑级海报美学。16:9。
```

### 雨夜高架夜市

```
赛博朋克雨夜高架夜市的设计图板，16:9横幅。
标题文字："霓虹兰花区 NEON ORCHID DISTRICT"。
分为五个不对称面板——一张大型电影级雨夜高架夜市场景，
两张原创成年赛博朋克信使近景肖像（颈部发光兰花纹身），
一张小型等距地图面板（小巷与无人机航线），
一张遗物面板（加密通行证/义体手套/贩卖机贴纸）。
分层使用霓虹洋红、青色、酸性绿，湿沥青反光，全息标牌。
密集但可读的构图，编辑级边距，小型标签。
```

### 室内科技空间

```
赛博朋克[空间类型]内部。
主光源为[方向]的[颜色]霓虹/屏幕光，
辅光源为远处透过[介质]的[颜色]环境光。
空间中有[科技设备]和[日常物品]的混合。
表面[材质描述]反射着[光源]的光斑。
空气中有薄雾/灰尘/蒸汽让光线可见。
35mm镜头，浅景深，前景元素虚焦框架。
```

## 核心约束

1. 必须有湿润表面——雨水/水洼/凝结水珠
2. 光源必须有可见来源——霓虹管/全息屏/机械灯/探照灯
3. 空气中必须有可见介质——雾/烟/蒸汽/灰尘
4. 建筑尺度要压迫——低角度仰拍、巨大结构、渺小人物
5. 禁止晴天/自然光/田园场景
