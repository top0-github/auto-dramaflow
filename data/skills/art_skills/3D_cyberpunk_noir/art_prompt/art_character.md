```

头微微转向镜头，眼神安静而挑衅。

```

```
- 多种表情变化

```

## 核心约束

1. 皮肤必须有真人质感——毛孔/细纹/汗珠/油光
2. 义体/装甲必须有使用痕迹——划痕/油渍/磨损
3. 光源必须有明确方向——霓虹灯管/全息广告/机械指示灯的反射
4. 禁止崭新无痕的科技设备
5. 禁止纯CG渲染感——需有胶片颗粒/变形镜头光斑/轻微色差
