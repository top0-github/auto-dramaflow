# 赛博朋克道具生成 prompt 规范

## 道具类型

### 武器

```
赛博朋克风格[武器类型]。
哑光[金属]与[聚合物]混合材质。
表面有使用痕迹——划痕/磨损/油渍。
[颜色]发光[元件]嵌入[位置]，在黑暗中发出柔和的[颜色]光。
散热口有轻微蒸汽。放置在[场景]中。
产品摄影布光——侧面硬光勾勒轮廓，正面柔光填充细节。
35mm镜头，浅景深，暗色背景。
```

### 科技设备

```
赛博朋克世界中的[设备类型]。
外壳为[材质]，表面有贴纸/编号/警示线。
[颜色]指示灯闪烁，小型全息屏幕投射[颜色]数据流。
连接线缆暴露在外，接口处有使用磨损。
放置在[场景]中，周围有[环境元素]。
```

### 日常物品

```
赛博朋克世界中的日常[物品]。
虽然是普通物品，但带有科技改造痕迹——[具体改造细节]。
表面材质混合了[传统材质]与[科技材质]。
使用痕迹明显——这个世界的物品都是用旧了的。
```

## 核心约束

1. 所有道具必须有使用痕迹——崭新的物品不存在于赛博朋克世界
2. 科技设备必须有可见的电源/线缆/指示灯
3. 材质优先选择：哑光金属/聚合物/碳纤维/回收塑料
4. 禁止纯装饰性发光——发光必须有功能来源（指示灯/屏幕/能源核心）
