# 分镜指导 — 动态少年漫

## 动漫动作词汇表 (Anime Action Vocabulary)

### 冲击帧 (Impact Frame)
单色关键帧插入打击命中瞬间。通常 1-3 帧。白色用于正能量/爆裂冲击，黑色用于暗黑力量/致命一击。画面内容为角色剪影 + 放射状速度线 + 纯色背景。目的：强化视觉冲击力，让观众生理性感知打击重量。

### 速度线 (Speed Lines)
从动作中心向外辐射的平行/放射线条组。密度对应动作强度。方向指示运动轨迹。在 MAPPA 风格中以锐利、不均匀间距排布，避免机械重复的等间距。线条粗细有变化，接近冲击点的线条粗，远离则细。

### 作画 (Sakuga)
高质量关键帧动画段落。动作流畅度远超常规镜头。通常用于：变身序列、必杀技释放、情感爆发、战斗高潮。特点是帧数充足、动作夸张、透视极端、阴影细节完整。在镜头序列中，sakuga 段落是资源分配的峰值点。

### 原画级线稿 (Genga-Quality)
关键帧线稿达到最终原画完成度。线条干净自信，无冗余笔触。线宽有变化（轮廓粗、内部细节细）。形体准确，透视正确。通常作为动画制作的基准帧。

### 赛璐珞阴影 (Cel Shading)
硬边阴影，无渐变过渡。明暗分界线清晰锐利。阴影形状经过设计，具有造型功能（不仅为真实感，更为构图美感）。通常使用 2-3 阶阴影（亮面、暗面、闭塞阴影）。彩色模式中阴影偏向深海军蓝而非纯黑。

### 边缘光 (Rim-Light)
从角色后方或侧后方打来的强光，在轮廓边缘形成高亮线条。用于将角色从背景中分离出来。在力量释放时刻，边缘光强度激增。彩色模式中使用电光青或暖金色边缘光。漫画模式中用白色留白线条替代。

## 战斗镜头结构 (Shot Structure for Battles)

### 序章 — 对峙
- **远景 (Wide/Long Shot)**: 展示战斗发生的完整空间，确立角色位置关系
- **中景 (Medium Shot)**: 双方对峙姿态，眼神交锋
- **特写 (Close-Up)**: 手部细节 — 握紧武器、捏碎物体、能量凝聚
- **极致特写 (Extreme Close-Up)**: 眼球反光中映出对手身影

### 起手 — 先手
- **低角度仰拍**: 先手攻击者从高处/远处发起攻势
- **速度线加速**: 移动轨迹
- **POV 主观镜头**: 从被攻击者视角看到攻击袭来
- **时间减速**: 关键时刻进入慢动作，展示细节

### 交锋 — 攻防
- **快速剪辑 (Fast Cut)**: 0.5-1 秒短镜头快速切换，展示高速攻防
- **匹配剪辑 (Match Cut)**: 上一镜动作方向与下一镜动作方向连续
- **反应镜头 (Reaction Shot)**: 围观者/同伴的震惊表情穿插
- **冲击帧插入**: 每一次重击命中配一个冲击帧

### 高潮 — 决胜
- **Sakuga 段落**: 终极技释放全流程
- **能量积蓄**: 粒子汇聚、光影渐强、速度线内收
- **释放瞬间**: 冲击帧（白色/黑色）
- **命中结果**: 慢动作展示伤害效果

### 落幕 — 余波
- **远景**: 战斗结束的完整场景，展示破坏范围
- **环境细节**: 飘落的碎屑、熄灭的火焰、断裂的武器
- **角色状态**: 疲惫姿态、受伤特写、眼神变化
- **情绪收束**: 从战斗的激昂过渡到战后沉思
