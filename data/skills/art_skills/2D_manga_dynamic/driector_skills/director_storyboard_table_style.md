# 分镜表风格 — 动态少年漫

## 漫画页面面板节奏 (Panel Rhythm for Manga Pages)

### 页面类型与面板分布

**开场跨页 (Opening Spread)**
- 双页跨页，无面板分割
- 大场景建立镜头，展示世界观
- 主角以剪影或背影出现在前景
- 日文拟声词（ドーン / ザワザワ）以大号书法字体跨页分布
- 画框出血（画面延伸至页面边缘，无边距）

**常规节奏页 (Standard Rhythm Page)**
- 5-7 个面板
- 面板大小节奏：小-中-大-小-中 → 制造阅读速度变化
- 大面板 = 慢阅读 = 重要信息
- 小面板 = 快阅读 = 过渡/反应
- 面板间距 (Gutter) 保持一致，2-3mm
- 倾斜面板边框用于动作场景，直线边框用于静态场景

**高潮页 (Climax Page)**
- 单页 1 大面板，占据 80% 以上页面面积
- 下方或侧方配 1-2 个反应特写小面板
- 主要面板画面出血
- 拟声词占据剩余空白区域
- 冲击帧美学在漫画中的运用：角色剪影 + 放射线 + 大面积网点填充

**落幕页 (Denouement Page)**
- 4-6 个等大小或渐小面板
- 从动作向静态过渡
- 最后一个面板为环境空镜
- 无拟声词
- 大量留白，创造呼吸空间
- 面板逐渐缩小 → 时间感逐渐拉长

### 开页构图

**左页 / 右页设计**
- 翻页前（左页）设置悬念/钩子
- 翻页后（右页）揭晓答案/冲击
- 重要揭示永远放在右页
- 利用翻页动作制造时间延迟（读者翻页 = 时间流逝）

**跨页动作场景**
- 攻击轨迹从左页跨越中线到右页
- 最关键打击点放在右页（翻页后第一眼位置）
- 中线处避免重要视觉信息（会被书脊遮挡）

## 日文拟声词运用 (Kanji Sound Effects)

### 拟声词分类
- **冲击系**: ドン (Don — 沉重打击), ドカッ (Doka — 爆裂冲击), ズドン (Zudon — 贯穿)
- **速度系**: ビュン (Byun — 高速掠过), シュッ (Shu — 快速移动), ザッ (Za — 瞬间冲刺)
- **氛围系**: ザワザワ (Zawazawa — 不安骚动), シーン (Shiin — 寂静), ゴゴゴ (Gogogo — 压迫感逼近)
- **能量系**: バチバチ (Bachibachi — 电光炸裂), ゴウッ (Gou — 火焰燃起), キラキラ (Kirakira — 闪耀)

### 拟声词设计规范
- 手写风格书法字体，融入画面构图
- 拟声词大小对应音量/强度
- 拟声词方向顺延动作轨迹
- 部分笔画可延伸为速度线
- 拟声词可作为面板间的过渡元素
- 彩色模式中，拟声词使用电光青或绯红色
- 漫画模式中，拟声词纯黑墨色，可叠加网点

## 色彩统一方案 (Color Unity)

### 彩色场景 (Color Scenes)
- **主色 Deep Navy (#0A1128)**: 夜空、阴影、深水、角色暗部
- **强调色 Electric Cyan (#00D4FF)**: 能量特效、边缘光、灵力、科技元素
- **爆发色 Crimson (#DC143C)**: 鲜血、火焰核心、愤怒标记、冲击高亮
- **辅助色 Warm Gold (#D4A574)**: 夕阳、回忆闪回、暖光源
- **中性色**: 角色肤色暖调、环境灰阶偏蓝

使用规则：
- 任何彩色画面中，至少出现主色 + 一个强调色
- 背景色温与前景角色分离（冷背景 + 暖角色 或 反之）
- 色调不超过 5 个主色系
- 饱和度控制在 80% 以下，保持"饱和但不霓虹"的质感
- 阴影不使用纯黑，偏向深海军蓝

### 漫画场景 (Manga Scenes — Black Ink + Screentone)
- **墨水**: 纯黑 (#000000)，用于主线条和实体阴影
- **网点纸 (Screentone)**: 
  - 10% 网点：浅灰阶，用于肤色阴影、远景色调
  - 30% 网点：中灰阶，用于衣物阴影、中等深度
  - 50% 网点：深灰阶，用于深色衣物、暗部
  - 70% 网点：极深灰，仅用于闭塞阴影和纯黑过渡
- **纸张底色**: 米白/泛黄 (#F5F0E8)，复古纸张质感
- **渐变网点**: 从高密度向低密度平滑过渡，用于天空、光晕、能量场
- **线网 (Line Screen)**: 平行线网点，用于金属表面反光、水面波纹
- **砂网点 (Sand Tone)**: 不规则颗粒网点，用于粗糙表面（岩石、树皮、布料）

使用规则：
- 禁止使用灰度渐变 — 所有灰阶必须通过网点实现
- 不同密度网点之间留 0.5mm 白边过渡，避免摩尔纹
- 速度线区域不使用网点
- 白色部分为纸张本色，不使用纯白填充
- 角色与背景通过网点密度区分（角色网点较浅，背景网点较深）
