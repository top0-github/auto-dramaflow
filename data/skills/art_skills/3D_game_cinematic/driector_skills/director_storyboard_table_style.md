# 导演分镜表格风格 - 3D游戏电影化

## 游戏类型HUD模板

### 黑暗奇幻动作RPG
- 左上角：生命条 (teal/rust配色) + 体力条
- 左下角：技能快捷栏 (4-6个技能图标)
- 右上角：小地图 (圆形，半透明)
- 右下角：当前任务目标文字
- 底部中央：Boss生命条（战斗中显示）
- 字体：衬线粗体，仿羊皮纸纹理背景

### 赛博朋克动作
- 左上角：全息生命条 (cyan/magenta发光)
- 左下角：义体能量条 + 快速破解菜单
- 右上角：增强现实小地图 (线框风格)
- 右下角：弹药计数 + 武器模式
- 中央：威胁指示器 (弧形)
- 字体：等宽无衬线，HUD元素带扫描线动画效果

### 史诗奇幻
- 左上角：队伍头像框 (gold/cream边框)
- 左侧竖排：角色状态列表 (HP/MP数值)
- 右下角：技能轮盘 (半透明)
- 顶部中央：罗盘导航条
- 字体：衬线优雅体，边框带石材纹理

### 像素复古RPG
- 四周：固定像素UI边框 (16-bit风格)
- 左侧竖排：角色肖像 + HP/MP条 (pastel配色)
- 底部：文字对话框 (位图字体)
- 右上角：金币计数 + 时钟图标
- 字体：8x8或16x16位图字体

### 低多边形策略
- 底部横条：单位信息面板 (warm earth tone)
- 右上角：资源计数器 (黄金/木材/石材图标)
- 左上角：回合计数器 + 菜单按钮
- 中央：选中单位的移动/攻击范围网格覆盖
- 字体：粗体无衬线，高对比度

## 色彩方案（按游戏类型）

### 黑暗奇幻

- 辅色：Rust (铁锈) #8B3A3A
- 强调色：Bone (骨白) #E8DCC8
- 环境：蓝调时刻 (blue hour) 冷色调

- 辅色：Magenta (品红) #FF00E5
- 强调色：Neon Yellow (霓虹黄) #FFFF00
- 环境：雨夜城市，湿润路面反光

## 屏幕截图真实性规则

为保证生成的图像具有真实的"游戏内截图"质感，必须遵循以下规则：

1. **轻微动态模糊**：角色动作和摄像机移动造成的自然拖影，不要完全锐利
2. **扫描线/像素网格**：复古和赛博朋克风格可选加 CRT 扫描线或轻微像素网格
3. **压缩伪影**：极轻微的 JPEG 压缩痕迹，模拟真实截图而非渲染输出
4. **HUD抗锯齿不完美**：UI元素边缘不做过度的亚像素渲染
5. **屏幕比例约束**：使用16:9或21:9的游戏标准宽高比
6. **景深效果**：摄像机焦点外的元素略带散景模糊
7. **色彩分级**：模拟游戏后期处理色调映射（LUT），不同场景使用统一调色
8. **禁止全屏完美构图**：避免电影海报式的完美构图，要有"玩家正在玩"的自然感
