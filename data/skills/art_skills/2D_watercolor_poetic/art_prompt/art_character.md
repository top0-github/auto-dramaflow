# 水彩人物生成 prompt 规范

## 人物模板

```
梦幻水彩人物画，[场景/氛围]。
[角色描述]——[性别][年龄]，[姿态/动作]，[服装——材质/颜色]。
印象派式水彩美学——松弛自信的笔触，透明的水洗。
色彩：柔和[色调描述]。
[光源方向/时间]的[光线质感]透过[环境元素]洒在[角色]身上。
冷压纸纹理可见——颜料在纸面纤维间自然扩散。
湿中湿（wet-on-wet）边缘柔和过渡，
干笔飞白（dry brush）用于[部位]的纹理。
画面保留大量留白——纸面的空白即是空气和光线。
编辑级高质量水彩插画。
```

## 角色范例

### 湖畔仕女
```
梦幻水彩插画，一位年轻女子穿着奶油色亚麻连衣裙，
坐在风化的石凳上，凝视着黄昏金色时刻的睡莲池塘。
印象派式水彩美学——松弛自信的笔触，
半透明的水洗以柔和的青绿与暖赭石色调呈现，
薰衣草灰的柔和阴影。
水面上睡莲和云朵倒影的柔焦。
温柔阳光透过垂柳枝条洒下。
冷压纸纹理贯穿可见——湿中湿边缘的晕染（bleeds），
精致的布光，大量留白的干净构图，
极简焦点于情绪——宁静、轻盈、转瞬即逝之美。
编辑级高质量插画，特纳水彩习作与现代编辑插画传统。
```

## 核心约束

1. 皮肤不能完全不透明——保留水彩的透明感
2. 面部特征用"暗示"而非"精描"——水彩不宜工笔
3. 服装褶皱用湿中湿晕染——不用硬线
4. 大量留白——纸面空白即是空气/光线
5. 禁止黑色/深灰不透明颜料——暗部用互补色混合（如群青+赭石）
