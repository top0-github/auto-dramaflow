# 水彩道具生成 prompt 规范

## 道具模板

```
水彩[道具类别]插画。
[道具名称]，[材质]以[水彩技法]表现。
色彩：[具体颜色描述]，半透明水洗层叠。
冷压纸纹理可见。
放置在[极简环境暗示]中。
柔和[方向]光，大量留白。
```

## 水彩技法速查

| 技法 | 效果 | 适用 |
|------|------|------|
| 湿中湿（Wet-on-wet） | 颜色柔和扩散、边缘模糊 | 花卉/布料/水面倒影 |
| 干笔（Dry brush） | 纸面纹理突出、颜色断裂 | 木纹/石面/旧物 |
| 渐变水洗（Graded wash） | 颜色从浓到淡均匀过渡 | 天空/墙面/光滑表面 |
| 留白液（Masking fluid） | 保留纯白区域 | 高光/水花/玻璃反光 |
| 盐粒效果（Salt texture） | 颜料干燥时形成雪花状纹 | 旧金属/石材/雪花 |

## 核心约束

1. 所有颜色必须是半透明——能看到纸面纹理
2. 高光用留白而非白颜料
3. 阴影用互补色层叠（如蓝+赭石得灰），不用纯灰/黑
4. 禁止不透明覆盖/数字填充
