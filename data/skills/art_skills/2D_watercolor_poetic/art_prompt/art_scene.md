# 水彩场景生成 prompt 规范

## 自然场景

```
水彩[场景类型]风景画，[时间/天气]。
[具体场景描述]。
透明水洗层叠构建空间——前景暖色（赭石+苔绿）、中景中调（鼠尾草绿）、远景冷调（薰衣草灰+淡青）。
湿中湿技法用于[云/水/雾]——颜料在湿润纸面自然扩散。
干笔用于[石/木/建筑]——纸面纹理透过断裂的笔触可见。
[光线描述]从[方向]来，透过[介质]洒在[表面]。
冷压纸面留白处即是[阳光/水光/空气]。
画面诗意、轻盈、有呼吸感。
```

## 场景范例

### 雨日植物温室
```
水彩雨日植物温室插画，清晨。
拱形玻璃温室的骨架，雨滴沿玻璃流下形成条纹。
悬挂的蕨类植物、兰花、陶土花盆。
一条狭窄的石板路，一张木制长椅上放着打开的园艺笔记本。
漫射的银色日光透过雨幕洒入。
透明水洗、颗粒化颜料、柔和的湿中湿晕染、可见的冷压纸纹理。
色调：鼠尾草绿、桉树灰、淡薰衣草、暖赤陶土、小面积黄色花朵点缀。
画面保持通透和诗意，保留纸面白色高光，
无硬边数码渐变，无照片级写实镜头效果，无浓重轮廓线。
```

## 核心约束

1. 必须保留冷压纸纹理可见
2. 高光 = 留白（非白颜料）
3. 远景比近景冷——大气透视的水彩版
4. 禁止浓重黑色轮廓线
5. 禁止不透明覆盖/照片级写实
