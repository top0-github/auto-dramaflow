# 分镜提示词风格 · 诗意水彩 · 技法参考

## prompt 结构

```
Style: Poetic watercolor animation, [具体氛围].
[镜头描述——含水彩技法].
Color: [透明色彩描述].
Texture: visible cold-pressed paper grain, preserved white paper highlights.
```

## 水彩动画关键词库

| 技法 | prompt关键词 |
|------|-------------|
| 湿中湿扩散 | `wet-on-wet bleeding`, `soft pigment bloom`, `color spreading on damp paper` |
| 干笔纹理 | `dry brush skipping across paper grain`, `broken stroke texture` |
| 渐变水洗 | `graded transparent wash`, `pigment fading from concentrated to paper-white` |
| 留白高光 | `preserved white paper highlight`, `reserved paper white for light` |
| 颗粒化 | `granulating pigment`, `sedimentary texture in damp area` |

## 情绪与色彩对应

| 情绪 | 主色调 | 技法倾向 |
|------|--------|----------|
| 宁静 | 鼠尾草绿 + 淡青 | 湿中湿为主、大量留白 |
| 温暖 | 暖赭石 + 暖金 | 干湿平衡、纸面暖调 |
| 忧伤 | 薰衣草灰 + 桉树灰 | 湿中湿为主、色彩柔和扩散 |
| 喜悦 | 淡珊瑚 + 暖金 + 嫩绿 | 干笔点缀花卉、纸面亮度提升 |

## 禁止

- 平滑矢量动画
- 不透明覆盖/白色颜料
- 硬边轮廓线
- 3D渲染/photorealistic
