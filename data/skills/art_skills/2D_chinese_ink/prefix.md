# 中国水墨画风 · 视觉风格手册

## 风格定位

中国传统水墨画（Ink Wash / 水墨）——以墨色浓淡层次构建空间、以留白制造想象、以笔触传递气韵的东方美学。借鉴范宽、马远等宋代山水传统，融合工笔细节与写意氛围。

## 核心美学

- **墨分五色**：焦/浓/重/淡/清——仅通过墨的浓度变化表现万物
- **留白即空间**：不画水而知水在、不画云而云自流——负空间是主动的构图元素
- **笔触即气韵**：每一笔都可见起笔/行笔/收笔的能量轨迹
- **纸为载体**：陈年宣纸的暖米色基调、纤维纹理可见、边缘自然毛边

## 色彩方案

| 元素 | 颜色 | 用途 |
|------|------|------|
| 主色 | 墨黑（多浓度层次） | 山/石/树/建筑轮廓 |
| 纸色 | 暖米色/陈年象牙 | 底色——不是纯白 |
| 点缀 | 朱砂红 | 印章（篆刻）、极小面积强调 |
| 点缀 | 花青/藤黄 | 极淡的设色——仅用于特定元素 |
| 月光 | 淡蓝灰 | 仅夜景 |

## 格式

- 立轴（竖幅 portrait）——山水画传统
- 手卷（横幅 landscape）——叙事性场景
- 扇面/册页——小品

## 禁止元素

- 浓重饱和的色彩（非水墨传统设色）
- 3D渲染/真实光影/物理精确透视
- 西方绘画的明暗法（chiaroscuro）
- 无笔触痕迹的均匀填充
- 纯白背景（必须是陈年纸色）
