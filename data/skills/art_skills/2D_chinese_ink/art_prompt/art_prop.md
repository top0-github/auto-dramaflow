# 水墨道具生成 prompt 规范

## 道具模板

```
中国传统水墨[道具类别]，画于陈年宣纸之上。
[道具名称]，[材质]质感以[笔墨技法]表现。
浓墨勾勒轮廓，淡墨渲染[部位]，飞白表现[纹理]。
放置于[场景暗示——极简，大量留白]。
纸面暖米色，纤维纹理可见。
左下角朱文小印。
```

## 常见道具笔墨处理

| 道具 | 笔墨技法 | 要点 |
|------|---------|------|
| 瓷器 | 细笔勾边 + 淡墨渲染釉面反光 | 釉面留白表现光泽 |
| 木器 | 干笔皴擦木纹 | 笔枯墨干才有木质感 |
| 铜器 | 浓墨勾边 + 淡墨渲染锈迹 | 锈迹用破墨法 |
| 书籍/画卷 | 淡墨染纸 + 浓墨写字 | 字迹可见但不需看清内容 |
| 兵器 | 铁线描勾刃 + 浓墨染柄 | 刃部留白表现寒光 |

## 核心约束

1. 道具在画面占比不超过 40%——大量留白
2. 必须有印章点缀
3. 禁止西方式产品摄影布光
4. 禁止纯白背景
