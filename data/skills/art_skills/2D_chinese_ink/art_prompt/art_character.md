# 水墨人物生成 prompt 规范

## 水墨人物模板

```
中国传统水墨人物画，[朝代]风格，画于陈年宣纸之上。
[角色描述]——[性别][年龄][身份]，[姿态/动作]。
面部以工笔（gongbi）精细勾画——[五官细节]，
衣纹以大笔写意（xieyi）扫出——墨色从浓到淡的飞白痕迹可见。
人物在画面中的占比为[比例]，其余为留白或淡墨环境暗示。
[道具/配饰]以浓墨勾勒，与衣纹的淡墨形成对比。
纸面有暖米色调，纤维纹理隐隐可见。
左下角红色朱文印章「[印文]」，右上角[位置]行书题字「[题字内容]」。
整体气质：[气质描述]。
```

## 角色类型范例

### 文人雅士
```
中国传统水墨人物画，宋代文人风格，陈年宣纸。
一位中年文士负手立于松树下，头戴幞头，身着宽袖长袍。
面部工笔精细——眉目清秀、神态安然、三绺长髯。
衣袍以大笔浓墨扫出，袖口飞白见笔力。
松树以浓墨皴擦，松针细笔勾画。
纸面暖米色，纤维纹理可见。
左下方朱文印章「清風明月」，右上方行书「松下行吟图」。
```

### 仕女
```
中国传统水墨仕女画，唐代风格，陈年宣纸。
一位年轻女子坐于窗前对镜梳妆。
面部以细笔工笔描画——柳叶眉、丹凤眼、樱桃小口。
发髻以浓墨渲染，发丝以游丝描勾画。
衣裙以淡墨大笔写意——纱质轻薄处留白、厚重处积墨。
铜镜以浓墨勾边，镜面留白——不画反影而反影自在。
朱砂色点缀：唇上一点红、发间一朵小花。
左下方朱文印章「花間」。
```

## 核心约束

1. 面部必须工笔精细——不可潦草
2. 衣纹必须见笔触——飞白/浓淡/干湿变化
3. 陈年纸色必须暖米——不可纯白
4. 必须有红色印章点缀
5. 禁止油画/水彩/素描等西方绘画质感
