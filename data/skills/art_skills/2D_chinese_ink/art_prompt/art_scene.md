# 水墨场景生成 prompt 规范

## 山水场景

```
中国传统水墨山水画，[朝代]风格（如范宽/马远/倪瓒），陈年宣纸。
层叠山峦通过墨色浓淡退入远方——
前景浓墨重笔（斧劈皴），中景中等墨色，远山几乎溶于淡灰雾气。
[建筑——亭/桥/寺/塔]置于山腰或水畔，
[人物]渺小地点缀其间——[动作]。
云雾以留白表现——纸面空白即是云海（留白负空间云法）。
松树以书法用笔勾枝——蟹爪枝/鹿角枝。
一道瀑布从中景跌落，以飞白笔法表现水流。
纸面暖米色，纤维纹理可见。
右上角草书题字「[题字]」，左下角朱文印章。
画面气质：[宁静致远/苍茫辽阔/孤寂超然]。
```

## 市井场景

```
中国传统水墨手卷（横幅），宋代风格，陈年宣纸。
[场景名称]的[时间]市集景象。
工笔级建筑细节结合写意水墨氛围：
[具体元素——桥/船/楼/摊/灯]。
小写意人物群像——[人物活动描述]。
可读的中国店铺招牌以行书写就：「[招牌内容]」。
墨色为主，点缀[灯笼赭色/月光淡蓝/朱砂印章]。
构图应读作连续的卷轴——人物群像的节奏性聚集与水面的留白交替。
避免现代物品、动漫面孔、过度饱和的海报式灯光。
```

## 核心约束

1. 留白是主动构图元素——不填满画面
2. 墨色必须有至少 3 个浓度层次
3. 必须有题字+印章
4. 纸色必须是暖米色（非纯白）
5. 禁止物理精确透视——使用散点透视或三远法（高远/深远/平远）
