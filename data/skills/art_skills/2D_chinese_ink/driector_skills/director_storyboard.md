# 分镜提示词风格 · 中国水墨 · 技法参考

## prompt 结构

```
Style: Traditional Chinese ink-wash animation, [具体风格标签].
[镜头描述——含笔墨技法].
Color: ink black + warm aged paper + vermillion seal [+ 淡设色].
```

## 笔墨技法词库

| 技法 | 效果 | prompt关键词 |
|------|------|-------------|
| 浓墨 | 近景/实体的重量感 | `bold dark ink`, `deep black strokes` |
| 淡墨 | 远景/虚化/氛围 | `pale wash`, `diluted grey ink` |
| 飞白 | 快速运笔的纹理 | `dry brush flying-white`, `broken stroke texture` |
| 破墨 | 湿墨破干墨的晕染 | `wet-on-dry ink bleeding`, `soft edge bloom` |
| 皴法 | 山石纹理 | `texture stroke (cun)`, `axe-cut stroke` |
| 工笔 | 精细勾线 | `gongbi fine-line`, `detailed brush outline` |
| 写意 | 大笔自由表达 | `xieyi freehand`, `expressive broad stroke` |

## 景别与笔墨对应

| 景别 | 笔墨处理 |
|------|----------|
| 远景山水 | 大写意泼墨——笔触大而自由 |
| 中景建筑 | 小写意——结构清晰但不过于工整 |
| 近景人物 | 面部工笔精细、衣纹写意 |
| 大特写 | 工笔极致——发丝/睫毛/指尖可见单笔 |

## 禁止

- smooth digital vector animation（平滑矢量动画感）
- 3D rendered ink simulation（3D仿水墨）
- 无笔触痕迹的均匀色块
