# 视听语言参考

## 三维坐标镜头系统

### Z轴（距离）—— 摄像机与被摄主体的距离

| 距离 | 范围 | 视觉效果 |
|------|------|----------|
| 极近 | 0-1m | 特写/大特写 |
| 近 | 1-3m | 近景/中近景 |
| 中 | 3-10m | 中景/中全景 |
| 远 | 10-30m | 全景 |
| 极远 | 30m+ | 远景/大远景 |

### Y轴（高度）—— 摄像机高度

| 高度 | 效果 |
|------|------|
| 极低（地面/仰拍） | 强化主体威严感 |
| 低（低角度仰拍） | 主体显得强大 |
| 平视（与主体同高） | 中性平等 |
| 高（高角度俯拍） | 主体显得渺小 |
| 极高（鸟瞰/顶视） | 上帝视角/全局 |

### X轴（方位）—— 摄像机水平位置

| 方位 | 角度 |
|------|------|
| 正面 | 0° |
| 斜侧面 | 45°/135° |
| 侧面 | 90° |
| 背面 | 180° |

## 镜头属性

| 属性 | 范围 | 说明 |
|------|------|------|
| 焦距 | 广角(<24mm) / 标准(35-50mm) / 长焦(85mm+) | 视野范围 |
| 光圈 | 大(f/1.4-2.8) / 中(f/4-5.6) / 小(f/8-16) | 景深控制 |
| 焦点 | 前焦/中焦/后焦/移焦 | 清晰区域 |
| 快门 | 正常(1/48s) / 慢速(<1/24s) / 快速(>1/1000s) | 动态模糊 |
| 帧率 | 常速(24fps) / 升格(60fps+) / 降格(<12fps) | 播放速度 |

## 景别体系

| 景别 | 英文 | 框取范围 | 叙事功能 |
|------|------|----------|----------|
| 远景 | Extreme Long Shot | 宏大场景，人物渺小 | 建立世界观 |
| 全景 | Long Shot | 人物全身+环境 | 交代空间关系 |
| 中全景 | Medium Long Shot | 膝盖以上 | 肢体语言 |
| 中景 | Medium Shot | 腰部以上 | 平衡人物与环境 |
| 中近景 | Medium Close-up | 胸部以上 | 表情和情绪 |
| 近景 | Close-up | 肩部以上 | 情绪强调 |
| 特写 | Extreme Close-up | 局部（眼/唇/手） | 极致细节 |

## 构图法则

| 法则 | 说明 |
|------|------|
| 三分法 | 画面九宫格，主体置于交点 |
| 黄金分割 | 1:1.618 比例 |
| 对称构图 | 平衡稳定，庄重感 |
| 引导线 | 线条引导视线至主体 |
| 框架构图 | 前景元素框住主体 |
| 负空间 | 大量留白，突出孤寂/渺小 |

## 空间层次

- **前景**：增加画面层次感和纵深感
- **中景**：主体所在的主要叙事空间
- **背景**：交代环境信息，营造氛围
