# 照片级写实角色提示词技法

> 来源：[awesome-gpt-image](https://github.com/ZeroLu/awesome-gpt-image)，社区实战验证

## 核心原则

**写实角色的 prompt 不是"描述一个美人"，而是"描述一张拍到了真人的照片"。**

- ❌ "一个漂亮的年轻女性，长发，微笑" → AI 味重、像摆拍
- ✅ "夏夜便利店门口，一个穿T恤的女生靠着冰柜看手机，玻璃反光映出她的侧脸" → 像摄影师抓拍

## 日常抓拍模式

适用于 `realpeople_modern_city` / `realpeople_urban_modern` 风格。

### 多人街头抓拍
```
夏夜晚上10点的便利店门口，3到4个年轻人在门口短暂聊天，
有人拿着饮料，有人坐在室外塑料椅上，有人站着看手机。
玻璃门窗内透出明亮白光，外面有暖黄色路灯和远处汽车车灯。
人物穿着日常服装：T恤、衬衫、短裤、牛仔裤、运动鞋。
不要网红造型。面部和姿态必须像真实路人，不能过度精修。
环境包含：冰柜贴纸、促销海报、垃圾桶、门口地垫、
玻璃反光、路边共享单车、地面上的饮料瓶水珠。
整张图应像摄影师在城市里抓拍到的真实生活切片。
```

### RAW iPhone 质感
```
完全 RAW 质感、未经处理、未经编辑、具备完整 iPhone 相机画质。
场景是[地点]，带有一瞬间的运动模糊。自然光，无闪光灯。
```

### 业余生活照
```
一张业余照片：[主体]在[场景]中[动作]。
从略高一点的角度拍摄，窗边自然日光，无闪光灯。
轻松的[环境]氛围，用 iPhone 拍摄。
```

## 角色一致性模式

适用于所有视觉手册的 `art_character.md` / `art_character_derivative.md`。

### 16格表情图
```
创建一张16格表情图，主角是一位[特征描述]的[角色类型]。
脸型、发型和服装必须在所有格子中保持高度一致。
16种表情应包含：开心、难过、生气、惊讶、害羞、无语、
邪魅一笑、沉思、好奇、自豪、委屈、不屑、困惑、害怕、哭泣，以及爱心表情。
```

### 官方角色设定参考页
```
基于这个角色和背景，创建一张类似官方设定资料的角色参考页。
- 包含三视图：正面、侧面和背面
- 添加角色的多种表情变化
- 拆解并展示服装与装备的细节部分
- 添加配色板
- 包含简短的世界观说明
- 整体采用规整版式（白底、插画风）
```

## 写实角色的核心约束

1. **不要"网红造型"** — 面部和姿态必须像真实路人，不能过度精修
2. **必须有环境上下文** — 角色不是在空白背景里，而是在真实场景中
3. **光源必须有来源** — 窗户光/路灯/顶灯/屏幕光，不能用"柔光"这种模糊描述
4. **服装必须日常** — T恤、衬衫、牛仔裤、运动鞋，不是礼服/汉服/西装（除非场景需要）
5. **要有"瑕疵"** — 略凌乱的头发、衣服褶皱、自然的阴影、不经意的表情
