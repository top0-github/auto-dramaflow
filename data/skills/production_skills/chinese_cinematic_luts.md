# 中式电影美学六调

## LUT-A · 茶橙酒楼暖光

适用：日常剧情、市井生活、暖心对话、家庭场景

| 参数 | 值 |
|------|-----|
| 色温 | 3000K（钨丝光） |
| 光比 | 1:4 |
| 颗粒 | 10% |
| 饱和度 | 中 |
| 对比度 | 中 |
| Bloom | 0.2 |

画面特征：暖橙色主导，如同老酒楼里钨丝灯泡的光，阴影温暖不冷硬，带着轻微的胶片颗粒。适合表现日常对话、市井烟火气。

---

## LUT-B · 粉绿花园柔光

适用：仙气独处、少女心事、治愈瞬间、浪漫初遇

| 参数 | 值 |
|------|-----|
| 色温 | 5200K（自然光） |
| 光比 | 1:3（柔光） |
| 颗粒 | 3% |
| 饱和度 | 中低 |
| 对比度 | 低 |
| Bloom | 0.35 |

画面特征：柔和的自然光 + 粉色/绿色偏移，画面干净通透，几乎没有颗粒感。适合表现治愈、温柔、初恋氛围。

---

## LUT-C · 金粉黄昏逆光

适用：宣传高光、主角出场、浪漫定格、英雄瞬间

| 参数 | 值 |
|------|-----|
| 色温 | 2900K（逆光） |
| 光比 | 1:6（高 bloom） |
| 颗粒 | 3% |
| 饱和度 | 中高 |
| 对比度 | 中高 |
| Bloom | 0.5 |

画面特征：强烈的金色逆光 + 镜头光斑，人物边缘被金光勾勒，背景略微过曝。适合主角出场的"决定性瞬间"。

---

## LUT-D · 青绿张艺谋重彩

适用：戏剧对峙、权谋冲突、高潮对手戏、仪式感场景

| 参数 | 值 |
|------|-----|
| 色温 | 4200K + 绿偏移 |
| 光比 | 1:5 |
| 颗粒 | 15% |
| 饱和度 | 高 |
| 对比度 | 高 |
| Bloom | 0.1 |

画面特征：厚重的青绿色调，高对比度的光影，浓烈的色彩饱和度。画面充满仪式感和戏剧张力，如同张艺谋式的视觉奇观。

---

## LUT-E · 雾紫凝眸肖像

适用：内心独白、人物志、情绪特写、心理戏

| 参数 | 值 |
|------|-----|
| 色温 | 5800K |
| 光比 | 1:2（平光） |
| 颗粒 | 5% |
| 饱和度 | 中低 |
| 对比度 | 低 |
| Bloom | 0.25 |

画面特征：柔和均匀的光线，淡紫色偏移带来一丝忧郁气质。人物面部细节丰富，背景虚化但保留色彩信息。适合深度的角色特写。

---

## LUT-F · 青赭文艺片油画

适用：文学瞬间、疲惫诗意、艺术片段、回忆闪回

| 参数 | 值 |
|------|-----|
| 色温 | 3600K |
| 光比 | 1:6（明暗） |
| 颗粒 | 20% |
| 饱和度 | 低 |
| 对比度 | 中 |
| Bloom | 0.15 |

画面特征：如同油画般厚重的质感，高颗粒感带来怀旧氛围，青赭色让画面蒙上一层时间的滤镜。适合文艺片质感的回忆/闪回段落。

---

## LUT 选择指南

| 短剧类型 | 推荐LUT |
|----------|---------|
| 都市爱情/甜宠 | LUT-B（粉绿柔光）或 LUT-C（金粉黄昏） |
| 古装权谋/宫斗 | LUT-D（青绿重彩） |
| 悬疑推理 | LUT-E（雾紫肖像）+ 暗部加深 |
| 文艺/独立短剧 | LUT-F（青赭油画） |
| 市井喜剧/家庭剧 | LUT-A（茶橙暖光） |
| 动作/热血 | LUT-C（金粉黄昏）+ LUT-D（青绿重彩）交替 |
