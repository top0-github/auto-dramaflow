# 生理细节描写标准

## 核心原则

**AI 视频模型对具体身体反应词（physiological cues）的执行力远高于抽象情绪词。**

- ❌ "很伤心" → 模型不知道该怎么拍
- ✅ "眼眶泛红 + 下唇咬紧 + 指节掐进掌心" → 模型稳稳执行

原因：视频模型训练数据中的情绪是通过具体身体反应表达的，不是通过抽象标签。

## 禁止的抽象情绪词

以下词语禁止单独出现在 prompt 中（必须替换或追加生理细节）：

| 禁止 | 推荐替换 |
|------|----------|
| 很伤心 | 眼眶泛红 + 泪水打转 + 嘴角颤抖 + 下唇咬紧 |
| 很生气 | 拳头攥紧 + 指节发白 + 太阳穴青筋微凸 + 牙关咬紧 |
| 很害怕 | 瞳孔收缩 + 呼吸急促 + 肩膀紧绷 + 后退半步 |
| 很幸福 | 眼角细纹 + 嘴角自然上扬 + 脸颊微红 + 眼睛发亮 |
| 很紧张 | 喉结滚动 + 鼻尖渗汗 + 手指微抖 + 吞咽动作 |
| 很疲惫 | 眼袋泛青 + 动作迟缓 + 肩膀下垂 + 说话有气无力 |
| 很梦幻 | 禁止使用。改为：柔光笼罩 + 发丝飘动 + 慢动作 + 背景虚化 |
| 很感人 | 禁止使用。改为：泪水滑落 + 嘴角颤抖 + 伸手想触碰又收回 |

## 生理细节映射表

### 面部

| 情绪线索 | 生理细节 |
|----------|----------|
| 压抑悲伤 | 眼眶泛红、下唇咬紧、嘴角微微抽动、眉心轻皱 |
| 克制愤怒 | 牙关咬紧、太阳穴脉搏可见、鼻翼扩张、眼睛眯起 |
| 恐惧 | 瞳孔放大、脸色发白、嘴唇发干、睫毛快速眨动 |
| 喜悦 | 眼角细纹、眼睛发亮、苹果肌上提、嘴角自然上扬 |
| 厌恶 | 鼻子微皱、嘴角下拉、眉头紧锁、眼睛斜视 |
| 惊讶 | 眉毛高抬、眼睛睁大、嘴巴微张、身体微微一僵 |

### 手部

| 情绪线索 | 生理细节 |
|----------|----------|
| 紧张 | 指节发白、手指微微颤抖、攥紧衣角、反复搓手 |
| 愤怒 | 拳头攥紧、指节咔咔响、青筋暴起、指甲陷进掌心 |
| 犹豫 | 手停在半空、手指蜷缩又松开、反复拿起又放下 |
| 决心 | 手慢慢攥紧、指节用力、一把抓住、不再犹豫 |

### 全身

| 情绪线索 | 生理细节 |
|----------|----------|
| 寒冷 | 肩膀缩起、手臂起鸡皮疙瘩、哈气可见、身体发抖 |
| 紧张 | 喉结滚动、吞咽动作、肩膀紧绷、呼吸急促 |
| 疲惫 | 步态沉重、肩膀下垂、头低垂、说话停顿增多 |
| 自信 | 脊背挺直、下巴微抬、步伐稳健、眼神不躲闪 |

## 约束规则

1. **每句 prompt 至少包含 1 个生理细节**（"分镜图辅助多参模式"和"首位帧模式"强制）
2. 生理细节必须是**可见的**（能呈现在画面上的），不能是内心描写
3. 动作必须是**具体微观的**（"指节发白"），不能是宏观概括的（"表现紧张"）
4. 生理细节与情绪必须**语义一致**（不能"眼眶泛红"配"她很高兴"）
5. 同时出现多个生理细节时，按面部→手部→全身的优先级排列
