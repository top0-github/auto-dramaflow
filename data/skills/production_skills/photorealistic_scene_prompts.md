# 照片级写实场景提示词技法

> 来源：[awesome-gpt-image](https://github.com/ZeroLu/awesome-gpt-image)

## 核心原则

场景的核心不是"有什么"，而是**"光从哪里来"**和**"谁的眼在看"**。

- ❌ "一个夜晚的城市街道" → 塑料感
- ✅ "夏夜晚上10点的便利店门口，玻璃门窗内透出明亮白光，外面有暖黄色路灯，远处有汽车尾灯拉出的红色光轨" → 真实

## 夜景模式

### 便利店/街头夜景
```
[时间]的[地点]，玻璃门窗内透出明亮白光，
外面有暖黄色路灯和远处汽车车灯。
路面有积水反光，空气中有一层薄雾。
环境包含真实细节：[具体店铺元素]。
整张图像摄影师在城市里抓拍到的真实生活切片。
```

### 雨夜
```
雨夜[地点]，湿润地面反射着霓虹灯光和车尾灯。
路灯的光晕在雨滴中扩散，远处建筑在雾气中模糊。
行人撑着伞匆匆走过，伞面上的雨滴在路灯下闪烁。
```

## 室内模式

### 自然光室内
```
[时间]的[房间类型]，窗边自然日光从[方向]照入，
无闪光灯，无人工补光。
[主体]在[位置/动作]。
画面有自然的阴影过渡，窗帘/百叶窗在墙上投下纹理。
用 iPhone 拍摄的日常感。
```

### 人工光室内
```
[房间类型]，唯一的[光源类型]在[位置]发出[色温]光。
其余空间沉浸在柔和的阴影中。
[主体]的脸被光源从[方向]照亮，另一半隐入暗部。
```

## 户外模式

### 黄金时刻
```
日落前最后一小时，金色阳光从低角度穿过[场景]。
长阴影拖在地面，[主体]的边缘被暖金色轮廓光勾勒。
天空是橙→粉→紫的渐变。逆光拍摄。
```

### 阴天/漫射光
```
阴天下午，柔和的漫射光从云层均匀洒下。
没有硬阴影，一切色彩都偏灰偏冷。
[场景细节]在这种光线下显得沉静而真实。
```

## 特殊模式

### 360全景
```
[地点]的 360 等距柱状全景图，完整展现[场景]全貌。
所有方向的光线一致，无缝拼接。
```

### 游戏截图感
```
[游戏名称] 游戏内画面，非常细致，非常真实。
镜头是对着一台固定的4K显示器近距离拍摄的，
画面带有轻微模糊，像是手持拍摄。
环境宽阔明亮，细节真实。
```

## 场景 prompt 核心约束

1. **光必须有来源** — 窗户/路灯/顶灯/屏幕/月亮，不能用"柔和的灯光"这种描述
2. **必须有"不完美"细节** — 垃圾桶/地垫/水珠/灰尘/磨损
3. **指定"拍摄方式"** — iPhone/单反/胶片/监控/手机前置，不同设备有不同的画面质感
4. **天气影响一切** — 晴/阴/雨/雾/雪不是装饰词，它们决定了反光率/阴影硬度/空气能见度
