# 提示词合规过滤规则

## 1. 硬性过滤（绝对拦截）

以下内容类别在 prompt 生成阶段即应过滤，不送入视频模型：
- 涉黄、色情内容及暗示
- 暴力、血腥、虐杀场景
- 惊悚恐怖元素（超出短剧审核底线）
- 敏感政治元素及隐喻

## 2. 隐性IP检测（重点警戒）

**极度警惕由通用词汇组成的电影/动画/游戏名称**——审核系统是机械的字符串匹配，不是语义理解。

| 高风险词组 | 触发原因 | 安全平替 |
|-----------|----------|----------|
| 奇幻森林 | 迪士尼电影名 | 发光巨型植物森林 |
| 冰雪奇缘 | 迪士尼电影名 | 冰雪覆盖的魔法王国 |
| 飞屋环游 | 皮克斯电影名 | 彩色气球托起的木屋 |
| 海底总动员 | 皮克斯电影名 | 珊瑚礁中的海洋生物群落 |
| 疯狂动物城 | 迪士尼电影名 | 拟人化现代都市动物社会 |

## 3. 中文分词误杀排雷

审核系统的字符串匹配会跨词边界触发：

| ❌ 高风险写法 | 触发原因 | ✅ 安全写法 |
|-------------|----------|------------|
| 废弃仓库里 | "库里"触发NBA球星IP | 废弃仓库内 / 废弃仓库中 |
| 库里放着 | "库里"触发 | 仓库内放置着 |
| 詹姆斯说 | "詹姆斯"触发NBA球星IP | 他说 / 角色名+说 |
| 黑色金刚石 | "金刚"触发电影IP | 黑色钻石 / 碳晶石 |

**原则：方位词后用"内"不用"里"。**

## 4. 双关词拆解

既是普通名词又是知名IP的词，必须物理降级拆解：

| 原词 | 风险 | 安全写法 |
|------|------|----------|
| 黑豹 | 漫威IP | 黑色豹子 |
| 毒液 | 漫威IP | 黑色粘液 |
| 金刚 | 电影IP | 巨型猩猩 |
| 闪电侠 | DC IP | 高速移动的身影 |
| 神奇女侠 | DC IP | 女战士 |

## 5. 生理不适词平替

| 可能触发风控的词 | 安全平替 |
|-----------------|----------|
| 虫子/昆虫群 | 发光生物 / 微小的光影 |
| 血液/流血 | 红色能量流体 / 深色液体 |
| 骷髅/白骨 | 几何结构 / 化石纹理 |
| 尸体 | 倒下的躯体 / 静止的身影 |
| 腐烂 | 风化的 / 时光侵蚀的 |

## 6. 输出纯净度

- 最终 prompt 输出前必须过一遍以上所有规则
- 每个被替换的词汇在日志中记录"原词→替换词→原因"
- 全局设定节中的风格描述也需过审
