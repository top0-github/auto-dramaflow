## 一、核心创作思想

### 1. 限制即力量

平面版画风格的核心哲学是**用最少的颜色表达最多的内容**。只允许红、黑、白三种颜色——红色是主角（占比最大），黑色提供结构和深度，白色作为留白和呼吸。这三种颜色的比例和位置决定了画面的全部张力和情绪。没有渐变可以过渡，没有阴影可以隐藏，每一笔都必须准确。

### 2. 几何化的世界

所有物体轮廓必须用**清晰的几何线**勾勒。不追求写实的体积感，而是追求平面的装饰性和秩序感。复杂形体被简化为最基本的几何元素组合——圆形、三角形、矩形。这种简化不是偷懒，而是提炼：去除一切非本质细节，只保留最核心的视觉识别特征。

### 3. 构图的力量

因为没有渐变和阴影来引导视觉，构图的平衡完全依赖**形状面积比和正负空间关系**。大面积红色块提供视觉重心，黑色线条勾勒边界和纹理，白色区域是视觉休息区。红色必须占比高于黑色，确保画面永远有"热"的核心。

## 二、美学特征

### 1. 配色铁律

| 颜色 | 角色 | 占比要求 |
|------|------|----------|
| 红色 | 视觉焦点、情绪载体 | > 黑色占比 |
| 黑色 | 轮廓、结构、纹理线条 | < 红色占比 |
| 白色 | 留白、背景、呼吸空间 | 无限制 |

**绝对禁止**：任何其他颜色、渐变、阴影、半透明。

### 2. 线条规范

- 所有元素必须有清晰几何轮廓线
- 线条粗细均匀，不使用粗细变化
- 轮廓线颜色：黑色
- 内部纹理线：黑色（如衣褶、木纹、砖缝）

### 3. 质感表达

- 2D 平面质感（无任何 3D 渲染感）
- 不使用任何材质纹理
- 所有表面为纯色填充
- 高对比度视觉冲击
- 极简平面版画质感

### 4. 适用场景

- 风格化角色海报
- 短剧标题卡/转场画面
- 极简主义品牌视觉
- 高冲击力封面设计
- 黑色幽默/讽刺类短剧的视觉基调

## 三、技术参考

| 参数 | 值 |
|------|-----|
| 渲染风格 | 纯 2D 矢量 |
| 配色模式 | 严格三色（Red/Black/White） |
| 色彩空间 | 纯色块，无过渡 |
| 轮廓线 | 均匀黑色，1-3px |
| 推荐输出格式 | PNG/SVG |
| 抗锯齿 | 关闭（保持像素级清晰边缘） |
