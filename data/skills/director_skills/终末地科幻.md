## 一、核心创作思想

### 1. 废土的浪漫：文明遗迹中的新生

终末地科幻的核心不是毁灭，而是**废墟之上的重建**。巨型能源塔在荒漠中运转、工业管线穿越大裂谷、运输轨道悬浮于废弃矿场上空——这些不是末日的遗骸，而是新文明的基础设施。画面中所有事物都带着"用过但仍在运转"的质感：金属表面有磨损但不破烂，建筑宏大但不奢华，科技先进但实用至上。

### 2. 探索者精神：冷静、专业、克制

这类作品的角色不是超级英雄，而是**工程师、探索者、战斗人员**——他们靠专业技能而非超能力解决问题。角色设计突出战术外套、工业机能服、局部装甲、工具腰包、能源核心挂件、机械义肢或外骨骼细节。气质上追求"冷静、专业、不花哨"，低饱和主色配少量高亮能量色。

### 3. 二次元游戏 PV 美学

镜头语言借鉴游戏宣传片的节奏：开场环境建立 → 角色登场 → 机械装置启动 → 战斗或动作爆发 → 能量特效高潮 → 标题收束。优先使用低机位仰拍、横向跟拍、推轨镜头、环绕镜头、快切镜头、慢动作镜头、景深镜头。

## 二、美学特征

### 1. 色调体系

- **基调**：低饱和蓝灰、青灰、黑蓝
- **能量点缀**：青色、橙色、紫色——只在高光/特效上出现，占比不超过画面 5%
- **肤色**：保持自然，不被整体冷调带偏
- **暗部**：保留层次，不死黑
- **后期**：轻微暗角、胶片颗粒、辉光、锐化

### 2. 场景构成法则

场景保持"**干净的大结构 + 少量高信息细节**"，避免过度杂乱：
- **大块体建筑**：巨型能源塔、工业管线、运输轨道、机械平台、冷色实验室、采矿设施
- **空间营造**：斜向构图、灯带、雾气、远景剪影
- **经典元素**：荒原基地、悬浮机械部件、高科技终端屏幕、风沙废土

### 3. 灯光设计

| 灯光角色 | 方案 |
|----------|------|
| 主光 | 冷色大面积柔光 |
| 边缘光 | 青蓝色或白蓝色勾勒角色轮廓 |
| 背景光 | 来自能源塔、屏幕或机械设备 |
| 点缀光 | 橙色警示灯、红色信号灯、蓝色能量灯 |
| 氛围光 | 体积雾隐藏远景光源，画面非均匀照亮 |

### 4. 材质与渲染

- 清晰二分阴影强化二次元感（NPR/EEVEE 工作流）
- 角色皮肤：柔和阴影 + 轻微高光
- 金属：降低真实反射，使用风格化高光
- 机械表面：边缘磨损、贴花、编号、警示线
- 能源核心：自发光材质
- 外轮廓：描边或反向法线描边
- 远景建筑：雾气降低细节压力

### 5. 特效设计

**原则："少而准"，每镜头只保留 1-2 个核心特效。**

可用特效：能源核心辉光、机械启动光圈、粒子尘埃、风沙流动、屏幕 HUD 线框、扫描光、低强度电弧、推进器尾焰、冲击波、镜头光斑、地面投影纹理。

## 三、技术参考参数

| 参数 | 建议范围 |
|------|----------|
| Bloom 强度 | 0.3-0.6（局部，不全局） |
| 体积雾密度 | 0.1-0.3（仅远景） |
| 描边粗细 | 1-2px（取决于分辨率） |
| 阴影软硬 | 清晰二分，过渡带宽 5-10% |
| 环境光强度 | 0.2-0.4（低环境光让主体突出） |
| 色温 | 5600-6500K（主光冷白） |
| 光比 | 1:4 到 1:6（高对比） |
| 颗粒 | 5-10%（轻微胶片感） |

### 检查清单

- 主体是否清晰
- 角色轮廓是否干净
- 冷暖对比是否明确
- 能量特效是否服务动作
- 场景是否有纵深
- 镜头节奏是否有起伏
- 画面是否具有二次元游戏 PV 感
