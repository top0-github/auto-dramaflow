## 一、核心创作思想

### 1. 尺度的敬畏

维伦纽瓦的科幻是一种**关于尺度的电影**。他的镜头永远在强化一个事实：在宇宙的宏大面前，人类是渺小的。巨大的沙虫、悬浮的飞船、无边际的沙漠——这些不是背景，而是角色。人物被挤压在画框边缘，或者像一粒沙子一样消失在画面中心。这种尺度感不是特效炫技，而是哲学追问：面对不可理解的力量，人如何保持尊严。

### 2. 二元对立的视觉系统

维伦纽瓦的色彩世界观总是**两种力量的对抗可视化**：
- 《沙丘》：厄拉科斯的橙金沙漠 vs 卡拉丹的冷蓝海洋
- 《银翼杀手2049》：橙黄的辐射废土 vs 青蓝的霓虹都市
- 《降临》：冷灰的军事基地 vs 温热的飞船内部

这种二元对立不是装饰，而是叙事核心的可视化。

### 3. 缓慢的力量

维伦纽瓦打破了"科幻片需要快节奏"的惯例。他的镜头移动极其缓慢——缓慢的推轨、缓慢的摇镜、缓慢的淡入。这种慢不是拖沓，而是**让观众的视线有足够时间浸入画面**，感受空间的重量、空气的湿度、沙粒的质感。

## 二、美学特征

### 1. 调色体系

| 元素 | 描述 |
|------|------|
| 主对比 | 橙金（暖） vs 冷蓝（冷）二元对立 |
| 饱和度 | 中高（但不溢色） |
| 暗部 | 深但不死黑，保留纹理 |
| 高光 | 温暖金黄或冷白，取决于场景 |

### 2. 镜头语言

| 技法 | 效果 |
|------|------|
| 大景深广角 | 展现宏大场景与人物渺小 |
| 缓慢推轨 | 逐步揭示空间规模 |
| 低角度仰拍 | 强化巨构建筑的压迫感 |
| 鸟瞰镜头 | 抽象化地形，展现几何美感 |

### 3. 光影特性

- **单一主光**：一个主光源统治整个画面
- **硬光**：明确的方向感和阴影
- **轮廓光**：人物边缘被光线勾勒
- **沙尘/雾气**：光线穿过大气介质形成可见光束

### 4. 场景设计语言

- 巨大简洁的几何形体（巨型建筑、飞船、设备）
- 极少装饰，功能至上
- 材质真实：混凝土、钢铁、沙石、玻璃
- 空旷空间 + 单个焦点

## 三、技术参考

| 参数 | 建议值 |
|------|--------|
| 色温 | 暖场景 3500-4500K / 冷场景 6500-8000K |
| 光比 | 1:4 到 1:8（高对比） |
| 画幅比 | 2.39:1 宽银幕 |
| 景深 | 大景深（f/8-f/16 等效） |
| 镜头焦段 | 18-35mm 广角为主 |
| 饱和度 | 中高（暖橙金/冷蓝各方向拉到极限但不溢出） |
